## Bootstrap

El comando para instalar es:
```sh
yarn add bootstrap@5.3.   @popperjs/core
```

luego, hay que importarlo en el main.jsx  así:

```sh
import * as bootstrap from 'bootstrap'
```

Y por último, tocar index.css dejando sólo
```sh
@import 'bootstrap'
```
